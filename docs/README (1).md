## ✨ Features

### 🤖 **Smart Chat Assistant**
- 💬 Interactive conversational interface
- 🧠 Context-aware responses
- 🎯 Multiple modes: General Questions, Coding Help, Project Queries
- 📜 Real-time message history with timestamps

### ✅ **Task Management System**
- ➕ Add, complete, and delete tasks
- 🎯 Priority levels (Low, Medium, High)
- 📅 Deadline tracking
- 📊 Task counter and visual status indicators
- ✓ Completion markers

### 📝 **Note Organization**
- 📄 Create and manage research notes
- 🗂️ Category-based organization
- 👁️ Live preview panel
- 🕒 Automatic timestamp tracking

### 🎨 **Beautiful User Interface - 3 Versions!**

| Version | Colors | Style |
|---------|--------|-------|
| **🌟 Light** | 3-4 | Clean & Simple |
| **💎 Enhanced** | 8-10 | Professional |
| **🌈 Vibrant** | 20+ | Rainbow Magic! |

## 🚀 Quick Start

### Prerequisites
- Python 3.7 or higher
- tkinter (built-in with Python)

### Installation & Run

```bash
# 1. Clone the repository
git clone https://github.com/Khan-Feroz211/AI-CHATBOT.git
cd AI-CHATBOT

# 2. Run your preferred version
python chatbot_demo_light.py          # Light version
python chatbot_enhanced_ui.py         # Enhanced (recommended)
python chatbot_vibrant_colors.py      # Vibrant (most colorful!)
```

**No additional dependencies required!** 🎉

## 💡 How to Use

### 1️⃣ **Chat Assistant**
- Open Chat tab → Select mode → Type question → Get instant help!
- Try: "show my tasks", "Python tips", "help me"

### 2️⃣ **Task Manager**
- Tasks tab → Enter task → Add → Track progress
- Visual indicators: 🔴 High, 🟡 Medium, 🟢 Low

### 3️⃣ **Notes**
- Notes tab → Add title & content → Save → Preview
- Categories: General, Research, Code, Ideas, References
